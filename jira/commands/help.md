# Jira Workflow - Quick Reference

## Available Commands

| Command | Description |
| ------- | ----------- |
| `jira:help` | This quick reference |

- `jira:start_ticket` - Start working on a Jira ticket

## Shared Commands

This plugin also includes shared commands for git operations:

| Command | Description |
| ------- | ----------- |
| `jira:commit` | Create a commit with user approval |
| `jira:describe_pr` | Generate PR description |
| `jira:create_worktree` | Create worktree for parallel work |
