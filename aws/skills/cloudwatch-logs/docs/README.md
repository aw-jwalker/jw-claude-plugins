# Documentation

Additional guides and reference materials for CloudWatch Logs.

## Purpose

Deep-dive documentation that supplements the main SKILL.md file.

## What Goes Here

- **filter-patterns.md** - Complete guide to CloudWatch filter pattern syntax
- **troubleshooting.md** - Common issues and solutions
- **performance-tips.md** - Best practices for efficient searches
- **aws-cli-reference.md** - Detailed AWS CLI commands reference

## When to Add Documentation

- You discover a CloudWatch feature not covered in SKILL.md
- You solve a tricky problem that others might encounter
- You learn a best practice worth documenting
- AWS releases new CloudWatch features

## Keep It Practical

Focus on:

- Real-world use cases
- Specific examples
- Lessons learned from experience
- AssetWatch-specific patterns
